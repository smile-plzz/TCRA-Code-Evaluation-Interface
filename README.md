# TCRA Code Evaluation Interface

## Overview
This is a dedicated, reviewer-facing research tool developed for an academic study on AI-generated code evaluation. The interface is designed to facilitate the systematic review of code artifacts generated by five different AI models (Gemini, ChatGPT, Claude Sonnet, Claude Haiku, and DeepSeek) across four distinct task domains (T1–T4).

## Evaluation Framework
The study utilizes the **TCRA Framework**, which evaluates artifacts across four critical dimensions:
- **Transparency**: Visibility of logic and rationale in the generated output.
- **Controllability**: The model's responsiveness to specific prompt steering and constraints.
- **Reliability**: Robustness and correctness across base tests and perturbations.
- **Auditability**: Traceability, provenance, and ease of verification/testing.

## Key Features
- **Centralized Artifact Explorer**: Sidebar navigation organized by Task and Tool for rapid context switching.
- **Dual-Pane View**: Parallel display of raw source code (with syntax highlighting) and live rendered previews for web artifacts.
- **Contextual Metadata**: Persistent display of the original input prompt, domain information, and study rationale for each specific artifact.
- **On-Demand Rubric Reference**: A persistent toggle to view the full TCRA scoring rubric and classification bands without losing the current review position.
- **Academic Aesthetic**: A clean, light-mode dashboard designed for high readability and professional technical review.

## User Instructions
1. **Navigate**: Select a Task (T1-T4) from the left sidebar and choose a specific AI tool.
2. **Review**: Examine the submitted prompt, study rationale, and the resulting code/preview in the main content area.
3. **Reference**: Open the "TCRA Rubric" using the button in the top navigation bar to assist in scoring.
4. **Record**: This interface is for **display only**. All scores should be recorded on the separate **printed scoring form (Sheet A1)**.

## Technical Details
- **Architecture**: Single Page Application (SPA).
- **Core Stack**: React, TypeScript, Tailwind CSS, Lucide icons.
- **Animation**: Motion for smooth drawer transitions.
- **Code Rendering**: Prism-based syntax highlighting for HTML, Python, and SQL.
